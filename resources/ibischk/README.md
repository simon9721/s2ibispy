# ibischk - IBIS Model Checker

## About
ibischk7 is the official IBIS model syntax checker and validator provided by the IBIS Open Forum.

## Version
- **ibischk7**: IBIS 7.x compatible checker

## Usage
The s2ibispy GUI will automatically detect and use `ibischk7.exe` from this directory if available.

You can also specify a custom path in the GUI's "ibischk7 Path (optional)" field.

## Download
If the executable is not included, download the latest version from:
https://www.eda.org/ibis/

## Platform Notes
- Windows: `ibischk7.exe`
- Linux: `ibischk7_linux` (mark as executable: `chmod +x ibischk7_linux`)
- macOS: `ibischk7_mac` (mark as executable: `chmod +x ibischk7_mac`)
